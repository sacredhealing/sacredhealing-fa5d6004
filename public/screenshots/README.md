# PWA Screenshots for Google Play

Add your app screenshots here for the PWA manifest and Google Play Store submission.

## Required images

### Wide (desktop/tablet) - 1280×720 px
| File | Screen |
|------|--------|
| `home-wide.png` | Home - Today's Sacred Practice |
| `meditations-wide.png` | Meditations page |
| `breathing-wide.png` | Breathing exercise |
| `healing-wide.png` | Healing audio |
| `music-wide.png` | Music - Browse tracks |
| `explore-wide.png` | Explore - Healing Journey |
| `community-wide.png` | Community - Chat & Go Live |
| `profile-wide.png` | Profile - Badges & streak |
| `membership-wide.png` | 30 Days of Sacred Healing |

### Narrow (phone) - 1080×1920 px
| File | Screen |
|------|--------|
| `home-narrow.png` | Home screen |
| `meditations-narrow.png` | Meditations page |
| `music-narrow.png` | Music |
| `profile-narrow.png` | Profile |
| `community-narrow.png` | Community |

## Format
- PNG or JPEG
- Max 8MB per image
- Min 320px on shortest side

## How to capture
1. Open your app in Chrome
2. Use DevTools (F12) → Toggle device toolbar for mobile size
3. Take screenshots of key screens
4. Resize to match dimensions above if needed
